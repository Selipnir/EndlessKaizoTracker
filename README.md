# Ironmon Tracker – Custom Extension & Ruleset

This repository contains a custom **Ironmon Tracker extension** along with its accompanying **ruleset**.

The extension is designed to be used with the Ironmon Tracker and BizHawk, adding additional logic and constraints to support this ruleset.

## Contents
- Tracker extension files
- Full ruleset documentation
- Open-source and free to modify

## Installation
1. Download the latest release
2. Copy the extension files into your `Ironmon-Tracker/extensions` folder
3. Enable the extension from within the Ironmon Tracker
4. The ReadMe in the extension has more information to help you get started
